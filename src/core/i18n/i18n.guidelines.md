### i18n guidelines (legacy core translations)

> **This directory is legacy and frozen for new keys.** It holds the MUI app's
> translations only. **New user-facing strings go to the CRD per-feature namespaces
> under `src/crd/i18n/<feature>/`** (all six languages — en, nl, es, bg, de, fr —
> edited in the same PR; managed manually, not Crowdin). See `src/crd/CLAUDE.md`.

For legacy upkeep of strings that already exist here:

- **Crowdin is no longer used.** Every locale file in this directory is now edited
  directly in the repo — `translation.en.json` **and** all non-English
  `translation.<lang>.json` files — in the same PR (the same manual, AI-assisted
  workflow as CRD).
- Do not add NEW keys here — add them to CRD instead (this directory stays frozen for new keys).
- When changing an existing string, update the English value and the matching value in
  each non-English file; **key parity across all languages must be preserved.**
