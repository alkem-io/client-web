### i18n

The project uses Crowdin as a translation provider.

Only the translation.en.json file is updated through the code. All other translations are managed through Crowdin.

##### How to get the latest translations

1. Go to Crowdin Dashboard (Alkemio project)
2. go to the Translations tab
3. Download as ZIP section
4. Build and download option
5. Then replace them in your local branch

##### If you need to restore translations

1. Go to Crowdin Dashboard (Alkemio project)
2. Find Pre-translation dropdown
3. Choose via Translation Memory
4. Confirm the options and start it
5. Once ready, you can build and download the updated translations from the Translations tab
