### i18n

**Crowdin is no longer used.** All translations are now maintained directly in the repository.

#### Legacy core (`src/core/i18n/`)

Serves the not-yet-migrated MUI app and is **frozen for new keys** — new strings go to CRD (see below). For upkeep of existing keys, edit the English source (`translation.en.json`) **and** the matching value in each non-English `translation.<lang>.json` file in the same PR. Key parity across all languages (en, nl, es, bg, de, fr) must be preserved.

#### CRD (`src/crd/i18n/<feature>/`)

All new user-facing strings live here, in per-feature namespaces, with all six languages edited in the same PR (manual / AI-assisted). Key parity is enforced in review (CodeRabbit). See `src/crd/CLAUDE.md` and `docs/crd/translations.md` for the full guide.
